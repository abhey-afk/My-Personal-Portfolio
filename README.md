## 🔧 Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/abhey-afk/My-Personal-Portfolio.git
   ```

2. Install dependencies:
   ```bash
   cd My-Personal-Portfolio
   npm install
   ```

3. Create a .env file:
   ```bash
   cp env.example .env
   ```

4. Start the development server:
   ```bash
   npm start
   ```
